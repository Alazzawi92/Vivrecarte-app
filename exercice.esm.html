Maîtrise JavaScript Moderne (ESM) appliqué à Express.js

Phase 1 — Modules ESM & Syntaxe Moderne (1–10)
01

Export Nommé

Structurer un module utilitaire réutilisable.
Créer un fichier math.js exportant une fonction add(a, b).
export const add = (a, b) => a + b;
Permet d’exporter plusieurs fonctions indépendantes et d’importer uniquement celles nécessaires.
02

Export par Défaut

Définir l’entité principale d’un module.
Exporter une classe Database en export par défaut.
export default class Database { connect(){} }
Un seul export par défaut par fichier, idéal pour services ou classes métier.
03

Alias d’Import

Éviter les conflits de noms.
Importer log sous le nom logger.
import { log as logger } from './utils.js';
Le mot-clé as permet le renommage stratégique.
04

Template Literals

Construire dynamiquement des chaînes.
Créer une URL dynamique avec id et slug.
const url = `/post/${id}/${slug}`;
Améliore la lisibilité comparé à la concaténation.
05

Destructuration Objet

Extraire proprement des données.
Extraire email depuis req.body.
const { email } = req.body;
Réduit la répétition et améliore la clarté.
06

Destructuration Tableau

Manipuler des collections efficacement.
Extraire les deux premiers éléments d’un tableau.
const [first, second] = tags;
Accès direct par position.
07

Spread Operator

Appliquer l’immuabilité.
Cloner un objet utilisateur et modifier son rôle.
const updated = { ...user, role: 'admin' };
Évite la mutation directe des objets.
08

Rest Parameters

Accepter un nombre variable d’arguments.
Créer une fonction de somme dynamique.
const sumAll = (...nums) => nums.reduce((a,b)=>a+b,0);
Regroupe les arguments dans un tableau.
09

Short-Circuit

Définir une valeur par défaut.
Assigner une valeur fallback si nom vide.
const name = user.name || 'Anonyme';
Utilise l’opérateur logique OR.
10

Optional Chaining

Sécuriser l’accès aux propriétés.
Accéder à user.address.city sans erreur.
const city = user?.address?.city;
Retourne undefined si propriété absente.
Phase 2 — Manipulation de Données JSON (11–20)
11

Filter

Filtrer un dataset.
Sélectionner les produits dont le prix est supérieur à 50.
const result = items.filter(p => p.price > 50);
Retourne un nouveau tableau filtré.
12

Map

Transformer un dataset.
Extraire les IDs d’un tableau d’objets.
const ids = users.map(u => u.id);
Produit un nouveau tableau transformé.
13

Find

Rechercher un élément unique.
Trouver l’utilisateur correspondant à un email cible.
const user = users.find(u => u.email === target);
Retourne le premier élément correspondant.
14

Some

Tester une condition partielle.
Vérifier si un utilisateur possède le rôle admin.
const hasAdmin = users.some(u => u.isAdmin);
Retourne un booléen.
15

Every

Valider une condition globale.
Vérifier que tous les produits sont en stock.
const valid = items.every(i => i.qty > 0);
Retourne true si tous satisfont la condition.
16

Sort

Ordonner un dataset.
Trier les utilisateurs par ordre alphabétique.
users.sort((a,b)=>a.name.localeCompare(b.name));
Tri mutatif du tableau.
17

Reduce

Agréger des valeurs.
Calculer le total d’un panier.
const total = cart.reduce((acc,c)=>acc+c.price,0);
Accumule une valeur unique.
18

Object.keys

Lister les propriétés d’un objet.
Récupérer les paramètres de requête.
Object.keys(req.query);
Retourne un tableau des clés.
19

Object.entries

Parcourir clé/valeur.
Transformer un objet config en tableau itérable.
Object.entries(config);
Permet une boucle dynamique.
20

JSON Parse/Stringify

Manipuler le format JSON.
Parser une chaîne JSON reçue.
const data = JSON.parse(jsonString);
Standard d’échange des API.
Phase 3 — Asynchronisme & Promesses (21–30)
21

Créer une Promise

Simuler un délai asynchrone.
const wait = ms => new Promise(r => setTimeout(r, ms));
22

.then()

Consommer une promise.
fetchData().then(res => console.log(res));
23

Async/Await

Réécrire avec await.
const res = await fetchData();
24

Try/Catch

Gérer une erreur asynchrone.
try { await fetch(); } catch(e) {}
25

Promise.all

Exécuter plusieurs requêtes en parallèle.
await Promise.all([f1(), f2(), f3()]);
26

Promise.race

Implémenter un timeout.
Promise.race([fetch(), timeout(5000)]);
27

Top-Level Await

Utiliser await à la racine d’un module.
const conn = await db.connect();
28

Fetch GET

Récupérer des données externes.
const res = await fetch(url);
29

Fetch POST

Envoyer un JSON vers un serveur.
fetch(url,{method:'POST',body:JSON.stringify(data)});
30

Async Map

Transformer un tableau via appels asynchrones.
await Promise.all(ids.map(id => fetchById(id)));
Phase 4 — Architecture Backend (31–40)
31

Higher Order Function

Créer une fonction retournant une fonction.
const multiplyBy = x => y => x * y;
32

Middleware Pattern

Modifier l’objet req avant next().
const auth=(req,res,next)=>{req.user='Bob';next();};
33

Closure

Créer un compteur encapsulé.
const counter=(()=>{let c=0;return()=>c++;})();
34

Dynamic Import

Importer conditionnellement un module.
if(dev) await import('./debug.js');
35

Map

Stocker des sessions via Map.
sessions.set(id,data);
36

Set

Supprimer les doublons.
const unique=[...new Set(ids)];
37

Erreur Personnalisée

Créer une classe ApiError.
class ApiError extends Error { constructor(status,msg){super(msg);this.status=status;} }
38

Regex

Valider un slug.
/^[a-z0-9-]+$/.test(slug);
39

Variables d’Environnement

Accéder à PORT.
const PORT=process.env.PORT||3000;
40

Factory Function

Créer un générateur CRUD.
const createCtrl=Model=>({list:()=>Model.find()});
